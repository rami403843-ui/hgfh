# Каталог игр на Django

Классная работа: мини-каталог любимых игр.

## Что реализовано
- Модель `Game` (название, жанр, год выпуска, отметка о прохождении) с методом `__str__`
- Миграции базы данных
- Регистрация модели в админ-панели Django
- View и шаблон: список игр выводится маркированным списком `<ul>`, при пустой базе — сообщение «Список игр пуст» через `{% empty %}`

## Как запустить
1. `python -m venv venv` и активировать окружение
2. `pip install django`
3. `python manage.py migrate`
4. `python manage.py runserver`
5. Открыть http://127.0.0.1:8000
